# Prism Engine

**Context Governance for LLMs** — Stop AI hallucinations by deploying a Context Server that forces coding assistants to follow your Design System.

## What is Prism Engine?

Prism Engine is an **MCP Server** (Model Context Protocol) that provides architectural rules, coding standards, and design constraints to AI coding assistants like:

- **Cursor**
- **Windsurf**  
- **Claude (via MCP)**
- **GitHub Copilot** (coming soon)

Instead of hoping your AI follows best practices, **Prism enforces them**.

## Quick Start

```bash
# 1. Clone and install
git clone https://github.com/jeffdev/prism-engine
cd prism-engine
npm install

# 2. Configure your connection
doppler setup  # Select your project

# 3. Seed your rules database
doppler run -- npm run db:seed -w packages/db

# 4. Start the MCP server
npm run dev -w apps/prism-mcp-server
```
